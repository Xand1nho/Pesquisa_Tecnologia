O que é HTML e para que serve?

Hyper Text Markup Language,em português chamada de Linguagem de Marcação de Hipertexto é a estruturação de uma página na web, onde, a escrita, imagens, links são criados nela, podendo ser utilizados na web. Ela se tornou a base de qualquer site de web, podendo ser decifrado pelos navegadores e exibir as informações de maneira visual para o usuário, o HTML é uma das linguagens de tecnologia principais do World Wide Web famoso www.

O que é CSS e para que serve?

Cascading Style Sheets, em português chamada de Folhas de Estilo em Cascata, é uma linguagem de estilização que é utilizada para a aparência visual dos elementos da página, como cores, fontes, espaçamento e layout. É a parte de deixar "bonito" mais organizado o navegador, deixa ele mais "limpo". O CSS é também é uma das principais para a Word Wide Web.

O que é JAVASCRIPT e para que serve?

JavaScript (frequentemente abreviado como JS), linguagem de programação utilizada na web que faz a interação do site, faz com o que o usuário possa interagir com o site de forma dinâmica e responsiva. Js faz com que um site faça sentido, que cause bastante impacto para o site.

O que é REACT e para que serve?

React é uma biblioteca do JavaScript que foi criada pelo famoso Facebook. Ela é utilizada para construir interfaces de usuário e especialmente para aplicações web de página úniuca. O React permite criar componentes reutilizáveis, assim facilitando o desenvolvimento e a manutenção do código.

O que é REACT NATIVE e para que serve?

React Native é uma biblioteca do JavaScript que permite criar aplicações móveis nativas usando React. Ela permite desenvolver aplicativos para iOS e Android com uma única base de código, facilitando o desenvolvimento e a manutenção.

O que é GITHUB e para que serve?

Github é uma plataforma de hospedagem de código que permite colaborar em projetos de software e controlar versões e gerenciar repositórios no Git.

O que é VITE e para que serve?

Vite é uma ferramenta de desenvolvimento web que oferece um ambiente de desenvolvimento rápido e eficiente para aplicações modernas, usamos o vite como um framework e também é bastante utilizade para frontend já que é rapida e facilita o desenvolvimento .

O que é EXPO REACT NATIVE e para que serve?

Expo React Native é um plataforma que facilita bastante o desenvolvimento de aplicativos moveis usando o React Native. Ela exlcui a necessidade de configurar ambientes nativos díficeis, já que vem com ferramentas prontas e ajuda basntate.

O que é Node_modules e para que serve?

Node_modules é a pasta gerada automaticamente pelo Node.js que contém todas dependências instaladas de um projeto. Ela é utilizada para armazenar bibliotecas e módulos necessários para o funcionamento de um projeto, sem ela basicamente não tem como rodar projeto.
